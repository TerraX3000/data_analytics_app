Purpose

This application serves as an in-progress prototype to demonstrate data analytic and data visualization methods.

Features

Data Visualization Samples

Includes a varierty of data visualization types created using the Dash and Bokeh visualization packages.

Data Manager

Provides the capability to upload new datasets to the SQL database as well as delete datasets.

Data Analyzer

Provides capabilities to analyze datasets which have been uploaded.

Research Info

Provides capability to capture online resources and references related to the purpose of this app.

Technical Description

Development

This software is written in Python and utilizes several packages including Flask (web framework), SQLAlchemy (database object-relational mapper), Pandas (data analysis), Dash (data visualization), and Bokeh (data visualization).

Web Hosting

The application is hosted on the Google Cloud Platform. It takes advantages of Google Cloud Run, a fully managed compute platform for deploying and scaling containerized applications quickly and securely.

SQL Database

The app relies on a MySQL database provided by Google Cloud SQL, a fully-managed database service that helps you set up, maintain, manage, and administer your relational databases on Google Cloud Platform. While the app directly connects to the MySQL database, it is also possible to connect to and administer the database using phpMyadmin, a free software tool written in PHP, intended to handle the administration of MySQL over the Web, and MySQLWorkbench, a unified visual tool for database architects, developers, and DBAs.
