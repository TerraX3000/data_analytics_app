Dataset Info provides a description of the datatypes in the dataset.

  • # - Integer-based index number of the column in the dataset
  • Column - the column name of the dataset field
  • Non-Null Count - the number of non-empty values in the column
  • Data Type:
    • int64 - Integer number type
    • float64 - decimal number type
    • object - a string or other non-number type

Description

The dataset analyzer will create a simple plot of the selected data column.

Instructions
  • Select the dataset on the dataset statistics tab
  • Choose a column from the Column Name menu below
  • Choose a plot type appropriate for the data type
  • Press Visualize Data Column to generate a plot
Limitations
  • Line graphs require numerical data
  • Bar charts require string (object) data
Example 1

The Titanic dataset contains information on the Titanic passengers. Several columns in this dataset contain numerial values which are suitable for plotting:

  • Index
  • PassengerId
  • Survived
  • Pclass
  • Age
  • SibSp
  • Parch
  • Fare
Example 2

The State Level Math Proficiency dataset containes education data from the Department of Education. Several columns in this dataset contain numerial values which are suitable for plotting:

  • Index
  • Data Group ID
  • Value
  • Number of Valid Test Takers

Note: Data visualization may take longer since this dataset contains over 50,000 records.
