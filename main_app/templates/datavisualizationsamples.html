Overview

The data visualizations shown on this page were created using the data visualization software packages Dash and Bokeh.

The code to create these plots is written in Python and viewable on Github here and here. In addition, Dash and Bokeh are compatible with the Pandas data analysis tool which is widely used by data scientists.

Dash

According to the Dash website, Dash is a productive Python framework for building web applications.

Written on top of Flask, Plotly.js, and React.js, Dash is ideal for building data visualization apps with highly custom user interfaces in pure Python. It's particularly suited for anyone who works with data in Python.

Through a couple of simple patterns, Dash abstracts away all of the technologies and protocols that are required to build an interactive web-based application. Dash is simple enough that you can bind a user interface around your Python code in an afternoon.

Dash apps are rendered in the web browser. You can deploy your apps to servers and then share them through URLs. Since Dash apps are viewed in the web browser, Dash is inherently cross-platform and mobile ready.

Bokeh

According to the Bokeh website, Bokeh is an interactive visualization library for modern web browsers. It provides elegant, concise construction of versatile graphics, and affords high-performance interactivity over large or streaming datasets. Bokeh can help anyone who would like to quickly and easily make interactive plots, dashboards, and data applications.

The code to create these plots is written in Python and viewable on Github here. Although the datasets in these examples are static, Bokeh is designed to support dynamic visualization using Python. In addition, Bokeh is compatible with the Pandas data analysis tool which is widely used by data scientists.

Cross Filter Example

Cross filters allow you to explore relationships between different attributes. This example includes economic, health, and environmental data for countries over several decades. As you hover over a point in the scatter plot, the time series graphs will update to show data for that country. You can also select different attributes from the dropdown menus, change the x-axis to use a log or linear scale, and use the slider to select a different year.

Real-Time Data Visualization

Data visualizaton dashboards can be constructed to display disparate data in frameworks to improve understanding of complex processes. This dashboard simulates real-time monitoring to implement statistical process control over manufacturing processes.
