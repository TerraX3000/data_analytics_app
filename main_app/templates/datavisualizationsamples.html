Overview

The plots shown on this page were created using Bokeh. According to the Bokeh website, Bokeh is an interactive visualization library for modern web browsers. It provides elegant, concise construction of versatile graphics, and affords high-performance interactivity over large or streaming datasets. Bokeh can help anyone who would like to quickly and easily make interactive plots, dashboards, and data applications.

The code to create these plots is written in Python and viewable on Github here. Although the datasets in these examples is static, Bokeh is designed to support dynamic visualization using Python. In addition, Bokeh is compatible with the Pandas data analysis tool which is widely used by data scientists.

Many of the plots below include a toolbar with tools to pan, zoom, select, and download an image of the plot. Bokeh supports many interactive features including sliders and dropdowns to use as data filters on dashboard visualizations.
